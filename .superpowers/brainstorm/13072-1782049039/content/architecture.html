🔑 关键设计决策

🎯 子域名路由

Next.js Middleware 检测 Host header，提取城市标识，注入到请求上下文。所有页面自动根据城市过滤数据。

例：jinan.sccoo.cn → 自动展示济南地区的招聘/转让/培训信息

🔐 第三方登录预留

OAuth 流程预留给 QQ/微信，先实现邮箱+手机号注册登录。NextAuth.js v5 的 Provider 机制天然支持后续扩展。

💳 支付预留

付费置顶功能前端+后端逻辑完整实现，支付环节抽象为接口，初期支持"手动确认"模式（管理员后台确认），后续接入微信/支付宝。

📱 PC + H5 同一套代码

Next.js 响应式设计 + TailwindCSS breakpoints，桌面端三栏布局，平板/手机自动切换为单栏卡片的移动布局。

📲 小程序独立开发

Taro 独立项目，通过 API 共享后端。小程序UI需要符合微信规范（如底部tabBar、原生导航栏），不适合直接复用Web组件。
